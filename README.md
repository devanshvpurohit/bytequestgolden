# Byte Quest: Dungeon of Binary

A 30-hour immersive, retro-themed hackathon hosted at the **Institute of Aeronautical Engineering (IARE), Hyderabad**.

## 🎮 The Experience
This project is more than just a landing page; it's an interactive journey. Participants must navigate a pixel-art platformer game, defeat the Glitch Overlord, and stabilize the system before they can enter the "Dungeon of Binary" (the hackathon).

## 🚀 Event Details
- **Date**: July 17 – 18, 2026
- **Duration**: 30 Hours
- **Venue**: IARE, Hyderabad
- **Prize Pool**: ₹1,20,000
- **Participants**: 280 – 480 Students (Teams of 1-4)
- **Registration Fee**: ₹799 per Team

## 🛠️ Tech Stack
- **Game Engine**: Phaser 3
- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion

## 📦 Getting Started

1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/Rahul-14507/qutammesh-hackathon.git
    ```
2.  **Install Dependencies**:
    ```bash
    npm install
    ```
3.  **Run Development Server**:
    ```bash
    npm run dev
    ```
4.  **Build for Production**:
    ```bash
    npm run build
    ```
