# 🚀 Quick Deploy to Vercel

## Instant Deploy (2 minutes)

1. **Visit Vercel**: https://vercel.com/new

2. **Import Git Repository**:
   - Sign in with GitHub
   - Click "Import Project"
   - Select: `devanshvpurohit/bytequestgolden`

3. **Auto-detected settings** ✅:
   - Framework: **Vite**
   - Build Command: `npm run build`
   - Output Directory: `dist`
   - Install Command: `npm install`

4. **Click "Deploy"** 🚀

5. **Done!** Your site will be live at:
   ```
   https://bytequestgolden.vercel.app
   ```
